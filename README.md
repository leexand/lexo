# Lexo

Modular Discord community assistant built for the lexoStudios ecosystem and InterWorld support infrastructure.

Lexo focuses on moderation, automation and community management through a scalable cog-based architecture.

---

## Features

- Moderation commands
- Event logging
- Captcha systems
- Community utilities
- Slash command support
- Modular cog architecture

---

## Tech Stack

- Python
- discord.py
- MongoDB
- dotenv

---

## Architecture

```txt
src/
├── core/
├── cogs/
│   ├── commands/
│   └── events/
├── models/
├── services/
└── utils/
```
